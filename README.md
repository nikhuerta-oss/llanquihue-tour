# LlanquihueTourApp


## Descripción

Sistema desarrollado en Java para gestionar tours turísticos.
Lee información desde un archivo txt, crea objetos y los almacena
en un ArrayList.


## Estructura

src/
- model/
- data/
- ui/

resources/
- tours.txt


## Ejecución

Ejecutar la clase:

ui.Main


El programa carga los tours y muestra los datos filtrados.
